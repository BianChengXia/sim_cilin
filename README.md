### 基于同义词词林的相似度计算

#### 测试
```
$simCilin = new SimCilin();
$text1 = '人民';

$sampleList = ["国民", "群众", "党群", "良民", "同志", "成年人", "市民", "亲属", "志愿者", "先锋" ];

foreach($sampleList as $text2) {
    $res = $simCilin->distance($text1, $text2);
    echo $text1. ' ' . $text2.' 相似度:' . $res . PHP_EOL;
}
```
